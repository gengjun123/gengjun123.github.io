1.1设备增加

request

POST /devices

{
  "name": "Sony/索尼 FDR-X3000R 潜水运动摄像机",
  "code": "KNHPIJ3488",
  "description": "摄像机...",
  "importTime": 1504512427148
}
名称 类型 描述 是否必须
name string 设备名称 是
code string 设备条码 是
description string 设备描述 否
importTime string 入库时间 否

成功 200

}
