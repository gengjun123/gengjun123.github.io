1.播放终端管理

1.1创建终端

request

POST /wxs/vr_clients

{
  "clientID": "fqj3fjp9ejfp9w8f983qf9q9p8",
  "cinameID": "eirt93q8jg9pw8j458398",
  "hallID": "fqofjp98jf9p8q3948fq3",
  "seatNumber": "02",
  "model": "vr0109",
  "MACAddr": "99:ed:9d:3b:e8:36",
  "IPAddr": "192.168.8.28",
  "description": "描述"
}
名称 类型 描述 是否必须
clientID string vr client id 是
cinameID string 影院id 是
hallID string 放映厅id 是
seatNumber string 座位号 是
model string 机器型号 是
MACAddr string 座椅主机的mac地址 是
IPAddr string 座椅主机的ip地址 是
description string 描述(最长265个字) 否

response

成功 200

{
  "clientID": "fqj3fjp9ejfp9w8f983qf9q9p8"
}

失败 400/500

{
  "info": "失败描述"
}

1.2修改终端

request

PUT /wxs/vr_clients/{client_id}

response

1.3删除终端

request

DELETE /wxs/vr_clients/{client_id}

response

2.影片管理

2.1创建影片

request

POST /wxs/movies

{
  "movieID": "q03jfp9q38jfp9q38fp9ew",
  "cinemaDistributedList": [{
    "cinemaID": "dfijq398f9q8f9q3w8f98q"
  },{
    "cinemaID": "fqiefiewngriw3nfrni83h"
  }],
  "name": "NAME",
  "movieCode": "movieCode",
  "duration": 30,
  "description": "简介",
  "type": "MOVIE",
  "category": "ACTION",
  "country": "USA",
  "iconURL": "asdfhf09q38fdhuf98q938f98pq",
  "posterURL": "g928gj98es9gw45jg98rsgfp9s8u",
  "scheduleStart": "1510208988793",
  "scheduleEnd": "1510208989793",
  "online": true,
  "weixinURL": "https://mp.weixin.qq.com/s?__biz=MzI5NjQ3OTg1Mw==&mid=2247483989&idx=1&sn=df852bf7447d293b6d6f34d5791a9d80&chksm=ec42ff57db3576413dbacf1288bc6be4d271a7db7aefc093e485c01026b3a645130e3e7b41a2&scene=0#rd",
  "price": "29.9",
  "isHot": true
}
名称 类型 描述 是否必须
movieID string 电影id 是
cinemaDistributedList list 已经分发的影院列表 是
cinemaDistributedList[].cinemaID list 影院id 是
name string 电影名称 是
movieCode string 电影编号 是
duration int 电影时长，单位是分钟 是
description string 电影简介 是
type string 类型枚举，有MOVIE(电影)，GUIDE(前导片)，AD(广告) 是
category string 电影分类枚举，有ACTION(动作)，SCIENCE(科幻)，等，可多传，用逗号(,)隔开 是
country string 电影发行国家枚举，有USA(美国)，CHN(中国)等 是
iconURL string icon http 地址 否
posterURL string poster http 地址 否
scheduleStart long 档期开始日期 否
scheduleEnd long 档期结束日期 否
online boolean 影片是否上线，默认false 否
weixinURL string 影片在微信上的简介 否
price double 价格 是
isHot boolean 是否热映 否

response

成功 200

{
  "movieID": "q03jfp9q38jfp9q38fp9ew"
}

失败 400/500

{
  "info": "失败描述"
}

3.点播单管理

3.1向点播单中添加影片(前端并无创建点播单操作，服务器会自动创建点播单)

request

PUT /wxs/orders/movie

{
  "movieID": "fqofnaieujfoiq3jfoaew",
  "userOpenID": "qjo3fjaefjap3jq83f9p8er9fpq4"
}
名称 类型 描述 是否必须
movieID string 所添加的影片id 是
userOpenID string 微信给用户对本公众号分配的唯一标识，获取方法参考 是

response

成功 200

{
  "orderID": "q03jfp9q38jfp9q38fp9ew"
}

失败 400/500

{
  "info": "失败描述"
}
